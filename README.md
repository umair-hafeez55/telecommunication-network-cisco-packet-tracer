# Telecommunication Network Design using Cisco Packet Tracer

## Overview

This project was developed as part of the Computer Networks course at Sukkur IBA University.

The objective was to design and simulate a secure enterprise telecommunication network using Cisco Packet Tracer. The network connects multiple departments across two floors while providing secure communication, resource sharing, dynamic routing, wireless connectivity, and VoIP services.

---

## Features

- Multi-floor enterprise network
- VLAN segmentation
- Inter-VLAN communication
- OSPF Dynamic Routing
- DHCP Configuration
- Wireless LAN
- VoIP (IP Telephony)
- Cisco ASA Firewall
- SSH Remote Access
- Port Security
- EtherChannel
- STP Configuration
- Network Testing & Verification
